# Account and Data Deletion Request

Last updated: June 2026

Rezeptmomente / Recipe Moments allows users to request deletion of their account and associated personal data.

To request account deletion, please send an email to:

appdev.jr2024@gmail.com

Subject:
Delete my Recipe Moments account

Please include:

* The email address used to create the account
* Confirmation that you want your account deleted

What will be deleted:

* User account information
* Authentication credentials
* Associated user-specific recipe data stored on our servers

Data stored only on your device can be removed by uninstalling the app.

Deletion requests are typically processed within 30 days.
